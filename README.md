# todoMongo
To do list with React as frontend and node and Mongodb as the backend
Make sure mongoDB is installed.

For frontend:
1. React-icons was used for the icons

For Backend:
1. mongoose for DB connection
2. helmetJs for headers 
3. express
4. nocache


## Installation
1. cd into client and run `npm i` then `npm start`
2. cd into server and run `npm i` then `npm run test`
3. visit `localhost:3000` to see the app.
